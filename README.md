<h1>Oil Shop POS System</h1>

<p>
This project is a Point of Sale system built for a small oil shop. It manages products, suppliers, employees, and sales. It also supports barcode functions and report generation. 
</p>
<img src="images/img.png" alt="Oil Shop Image" style="max-width:200px; height:auto;">
<h2>Features</h2>
<ul>
    <li>Add, edit, and delete products</li>
    <li>Manage suppliers and employees</li>
    <li>Record daily sales</li>
    <li>Generate PDF bills using ReportLab</li>
    <li>Create and scan barcodes</li>
    <li>Export data to Excel</li>
    <li>Simple Flask based web interface</li>
    <li>Secure login system with Flask Login</li>
    <li>Environment variable support with dotenv</li>
</ul>

<h2>Technologies Used</h2>

<h3>Backend</h3>
<ul>
    <li>Python</li>
    <li>Flask</li>
</ul>

<h3>Database</h3>
<ul>
    <li>MySQL</li>
</ul>

<h2>How to Run</h2>
<ol>
    <li>Clone the repository</li>
    <li>Create and activate a virtual environment</li>
    <li>Install dependencies<br>
        <code>pip install -r requirements.txt</code>
    </li>
    <li>Configure the database in the <code>.env</code> file</li>
    <li>Run the application<br>
        <code>python app.py</code>
    </li>
    <li>Open the browser and go to <code>http://127.0.0.1:5000</code></li>
</ol>
